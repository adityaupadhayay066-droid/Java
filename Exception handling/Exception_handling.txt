Exception Handling - It is a machenism in java that handle run time error so that your programm dont crash it run smoothly 
Important keyword of Exception Handling
1. try
Code that may cause exception
2. catch
Handles the exception
3. finally
Always executes (whether exception occurs or not)
4. throw
Used to manually throw an exception
5. throws
Declares exceptions in method
